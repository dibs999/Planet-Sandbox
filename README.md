# Planet Sandbox

Ein experimentelles 2D-Pixel-Sandbox-Spiel direkt im Browser. Zeichne Elemente auf eine kleine Welt, experimentiere mit eigenen Kombinationen und beobachte, wie Menschen und Aliens eigenständig Ressourcen sammeln und Gebäude errichten.

## Features

- Mehrere Planeten mit unterschiedlichen Startbedingungen (Terra, Pyros, Aquarea, Xylon)
- Große Auswahl an Elementen wie Wasser, Lava, Holz, Metall, Dampf, Rauch und mehr
- Keine vordefinierten Reaktionen – alle Elemente bleiben stabil, damit du selbst Verbindungen ausprobieren kannst
- Autonome Menschen, die Holz sammeln, Bäume pflanzen und Häuser errichten
- Fremde Aliens, die Kristalle abbauen und Biokuppeln bauen
- Anpassbare Pinselgröße und -dichte sowie Pausen- und Schrittmodus zum Beobachten der Simulation
- Mobile Ansicht mit einblendbarem Steuerungsmenü für komfortables Spielen auf dem Smartphone

## Nutzung

Öffne einfach `index.html` in einem modernen Browser. Für die beste Erfahrung sollte JavaScript aktiviert sein.

## Entwicklung

Die wichtigsten Dateien sind:

- `index.html` – Grundgerüst und UI-Struktur
- `styles.css` – Retro-inspirierte Oberfläche
- `script.js` – Spiel-Logik mit Element-Simulation und KI

Viel Spaß beim Erschaffen deiner eigenen Welten!
